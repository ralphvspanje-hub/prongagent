# User Files

Place your personal career files here. Skills reference this folder for source documents.

## Suggested files

| File | Used by | Purpose |
|------|---------|---------|
| Your resume (PDF) | career-mentor | Resume tailoring, cover letter writing |
| CV tailored versions (.md) | career-mentor | Track different CV versions per role type |
| LinkedIn profile export (PDF) | career-mentor | LinkedIn audit and optimization |
| Win log (.md) | win-log, career-mentor | Full achievement record |
| STAR story bank (.csv) | win-log, mock-interview | Interview story preparation |
| Technical prep notes (.md) | interview-prep | Technical readiness tracking |
| Cover letters (.md/.pdf) | career-mentor | Past cover letter reference |

## Job tracker

The `job-tracker/` subfolder is where the job-scan skill looks for your job tracker file. If you use an external job scanning agent, configure it to write here.

Configure the tracker path in `config/settings.md`.

## Privacy

This folder is gitignored in the template. Your personal files stay local.
